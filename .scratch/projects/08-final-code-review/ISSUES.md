## Severity Levels

- **Critical**: Blocks functionality, security risk, or data loss
- **High**: Major functionality impaired, workarounds difficult
- **Medium**: Functionality affected, workarounds exist
- **Low**: Minor issues, cosmetic, or nice-to-have improvements

## Known Areas to Check

### Potential Issue Areas

### Preventive Questions
- Are there any TODO comments in the code?
- Are there any unhandled exceptions?
- Is there dead code that should be removed?
- Are imports properly organized?
